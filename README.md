## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/saidsaidchiichii-coder/IlyassAgentAI.git

# Install dependencies
npm install

# Set environment variables
cp .env.example .env
# Add your GROQ_API_KEY, Firebase config, Stripe keys

# Deploy to Vercel
vercel deploy
```
